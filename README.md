# Хочу Суши 3D — GitHub Pages версия

Готовая версия для GitHub Pages. Внутри нет внешних CDN и Three.js: минимальный WebGL2-движок встроен в `index.html`, поэтому проект должен запускаться с GitHub Pages без дополнительных библиотек.

## Файлы

- `index.html` — игра.
- `assets/player_model_optimized.glb` — оптимизированная версия загруженной GLB-модели, используется как главный персонаж.

Оригинальная модель была около 91 МБ, для мобильного браузера она слишком тяжёлая. В этой версии модель уменьшена примерно до 1.8 МБ: сохранена форма и текстура, но снижена плотность сетки и размер текстуры.

## Как загрузить на GitHub Pages

1. Создай репозиторий, например `hochusushi3d`.
2. Загрузи в корень репозитория:
   - `index.html`
   - папку `assets`
   - `README.md`
3. Открой **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main`, folder: `/root`.
6. Ссылка будет вида `https://твой-ник.github.io/hochusushi3d/`.

## Управление

- iPhone/Android: наклон влево/вправо.
- Тап или удержание: броски васаби.
- ПК: A/D или стрелки, Space.
